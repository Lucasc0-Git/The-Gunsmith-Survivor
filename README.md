**A top-down survival shooter with crafting, base-building, and resource management.**  
Protect *The Core* while scavenging, crafting weapons, building defenses, and surviving waves of enemies in a hostile world.

**Singleplayer experience** — my first major Godot project, currently in active development. Made with **Godot 4+**.

## 🎮 Features

- **Top-down twin-stick style gameplay** with smooth movement and aiming
- **Day/Night cycle** with increasing difficulty at night
- **Survival mechanics**: Scavenge resources, mine materials, craft items
- **Crafting & Smithing**: Build weapons, tools, and structures at stations
- **Base building**: Place defenses, torches, fireplaces, and more to protect The Core
- **Inventory system** with stacking, hotbar, and drag & drop
- **Procedural elements** & enemy waves (zombies and more)
- **Polished HUD**, options menu, pause, and game over screen

## 🚀 How to Play / Download

**Recommended**: Download a pre-built version from the [Releases tab on GitHub](https://github.com/Lucasc0-Git/The-Gunsmith-Survivor/releases) or from [Itch.io](https://your-itch-page.itch.io/the-gunsmith-survivor) (Windows & Linux builds).

Alternatively:
1. Clone the repository:
   ```bash
   git clone https://github.com/Lucasc0-Git/The-Gunsmith-Survivor.git

Open the project in Godot 4+:
Navigate to the Godot-Project-Files folder
Open project.godot

Export the game using Godots export templates (File > Export) for your platform (Windows or Linux recommended).

Controls (default):

WASD — Move
Mouse — Aim
Left Click — Shoot
E — Toggle Inventory
Mouse Wheel — Switch weapons
1-5 — Hotbar slots
Space — Dodge (or assigned key)
F11 — Toggle Fullscreen

Check Input actions in Project Settings for full mapping.
